# AI-Eyewear

**AI-Eyewear** (operated by Even Realities Switzerland) maintains open,
verifiable references about AI glasses. We are an authorized Even Realities
reseller. AI-Eyewear resells the Even G2; it does not manufacture the other
products described in the database.

## Open resources we maintain

- **AI Glasses Open Database** — an open, machine-readable dataset of AI glasses,
  published as JSON and CSV under CC BY 4.0 (code under MIT).
  Repository: https://github.com/ai-glasses-eyewear/ai-glasses-eyewear

## Our commitment to verified data

- **Null, not guessed.** Where a value is unknown, we record it as `null` rather
  than inventing a number.
- **Sourced.** We prefer official manufacturer specifications and cite the data
  version and date for every published figure.
- **Versioned & citable.** Releases are versioned and include a `CITATION.cff`
  file so a specific snapshot can be referenced.

## How to contribute

- Open an issue or pull request with a proposed value, the product and field it
  applies to, and a verifiable source URL.
- Corrections and additions are welcome; unknown fields stay `null` until a
  source is provided.
